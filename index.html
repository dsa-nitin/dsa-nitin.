#01 ARRAY

What is an Array?

Answer: Array ek linear data structure hai jo elements ko contiguous memory mein store karta hai.
Complexity: Access O(1), Search O(n).
#02 STACK

What is a Stack?

Answer: Stack ek LIFO (Last-In-First-Out) data structure hai. Isme insertion aur deletion ek hi end (Top) se hota hai.
